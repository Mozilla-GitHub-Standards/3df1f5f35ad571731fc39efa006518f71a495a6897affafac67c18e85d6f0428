**Working Prototype Prizes:**  
$400,000 – First Place  
$250,000 – Second Place  
$100,000 – Third Place  
$50,000 - Fourth Place  

**Documentation might include:**
+ Network test data  
+ Video demos  
+ Photographic documentation  
+ Software code  
+ Hardware schematics
